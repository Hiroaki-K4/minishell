# How to build
Build the files.

```
make
```
Rebuild
```
make re
```
Clean .o files
```
make clean
```
Clean all files
```
make fclean
```

# How to run
First, launch minishell with the following command.
```
./minishell
```
Please insert something shell command and push enter.
```
minishell> ls
```
